What is a table in HTML

A table is a structured set of data made up of rows and columns (tabular data). A table allows you to quickly and easily look up values that indicate some kind of connection between different types of data, for example a person and their age, or a day of the week, or the timetable for a local swimming pool.

Table Tags

<tr> Table Row Element

The table row element, <tr>, is used to add rows to a table before adding table data and table headings.

<td> Table Data Element

The table data element, <td>, can be nested inside a table row element, <tr>, to add a cell of data to a table.

Code Example

<td> Table Head Element

The table head element,<thead>, defines the headings of the columns encapsulated in table rows.

Code Example

rowspan Attribute

Similar to colspan, the rowspan attribute on a table header or table data element indicates how many rows that particular cell should span withing the table.
The rowspan value is set to 1 by default and will take any positive integer up to 65534.

Code Example

<tbody> Table Body Element

The table body element,<tbody>, is a semantic element that will contain all table data other than table heading and table footer content. If used, <tbody> will contain all table row <tr> elements, and indicates that <tr> elements make up body of the table.
<table> cannot have both tbody and <tr> as direct children.

Code Example

<th> Table Heading Element

The table heading element, <th>, is used to add titles to rows and columns of a table and must be enclosed in a table row element, <tr>

Code Example

colspan Attribute

The colspan attribute on a table header <th> or table data <td> element indicates how many columns that particular cell should span within the table.
The colspan value is set to 1 by default and will take any positive integer between 1 and 1000.

Code Example

<tfoot> Table Footer Element

The table footer element, <tfoot>, uses table rows to give footer content or to summarize content at the end of a table.
